# OrbitaAlerta Mobile

App React Native + Expo — dashboard offline-first da Global Solution.

## Início rápido

```bash
npm install
npm start
```

Documentação completa: **[docs/ORBITALERTA_MOBILE.md](docs/ORBITALERTA_MOBILE.md)**

## Integração com API

Copie `.env.example` para `.env` e defina `EXPO_PUBLIC_USE_MOCK=false` quando o backend estiver pronto.
